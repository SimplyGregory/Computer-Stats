# Computer-Stats 👋

Python + Tkinter desktop UI that displays live computer stats (CPU/RAM and related system info) in a clean, compact overlay.
Built as a practical lightweight stats HUD using `psutil`.

## About This Project
<hr />

- 🖥️ Shows live system stats in a small always-on-top style window.
- ⚡ Updates quickly without heavy dependencies.
- 🧩 Includes two variants/scripts you can run depending on the UI you want.

## Files
<hr />

- `CmpStats1.py` — UI variant #1
- `CmpStats2.py` — UI variant #2 (includes CPU/RAM via `psutil`)

## Run
<hr />

Install dependency:
- `pip install psutil`

Run a variant:
- `python CmpStats2.py`
- `python CmpStats1.py`
